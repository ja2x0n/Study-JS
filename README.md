# JavaScript Study

React 개발을 위한 JavaScript 학습 레포입니다.

프론트엔드 개발에 필요한 JavaScript 기본 문법부터  
함수, 객체와 배열 처리, 비동기 처리, API 통신 등  
React 개발에서 자주 사용하는 JavaScript 개념을 직접 작성하며 학습합니다.

단순히 문법을 정리하는 것이 아니라,  
다양한 예제를 구현하고 데이터를 직접 다루면서  
JavaScript를 활용하는 능력을 키우는 것을 목표로 합니다.
